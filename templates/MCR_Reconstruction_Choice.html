Description

1) Correlation Functions :Reconstruction using correlation functions employs the Yeong-Torquato method which involves Simulated annealing based random pixel swapping to match the correlation function of reconstructed image to that of input image. The choice of correlation function to be matched and the number of reconstructed images can be chosen by user.

2) Physical Descriptors (Coming Soon!) :Reconstruction methodology will depend on the descriptors chosen. This webtool follows a four step hierarchical methodology with descriptors at higher scale being considered first. Please refer to articles by Xu et.al. (see references) for more details.

3) Spectral Density Function (Coming Soon!) :The reconstruction method depends on the type of microstructure. For channel type microstructure, a level cut Gaussian random field is used. On the other hand, a random disk packing algorithm is employed for microstructures with disk shaped filler. Please refer to article by Yu et.al.(see references) for more details.

Choose the characterization method from the options below.

NOTE: Images must be binarized
